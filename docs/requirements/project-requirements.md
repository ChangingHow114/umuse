# UMuse — 项目需求文档

## 项目概述

**项目名称**: UMuse
**项目类型**: 桌面应用（Windows）
**目标用户**: 音乐爱好者、乐手、编曲学习者
**核心价值**: 一键从混合音频中还原乐谱、音色和效果器参数

## 功能需求

### F1 — 音频分轨 (Stem Separation)
- **输入**: MP3/WAV/FLAC 混合音频文件
- **输出**: ≥5 轨独立乐器音频 (WAV)
- **分轨目标**: Drums / Bass / Vocals / Guitar / Piano / Other (6轨)
- **技术**: Demucs htdemucs_6s 模型

### F2 — MIDI 转录 (MIDI Transcription)
- **输入**: 单轨乐器音频 (WAV)
- **输出**: 标准 MIDI 文件 (.mid)
- **支持乐器**: 钢琴、吉他、贝斯、鼓、人声
- **要求**: 音符起止时间、音高、力度

### F3 — 乐谱生成 (Sheet Music Generation)
- **输入**: MIDI 文件
- **输出格式**:
  - 简谱（数字谱 / 字母谱）
  - 五线谱（标准钢琴谱）
  - 六线谱（吉他 TAB 谱）
  - 乐队总谱（多乐器合谱）
- **要求**: 自动检测调号、拍号，支持 PDF/PNG 导出

### F4 — 音色拟合 (Timbre / Preset Matching)
- **输入**: 分离后的单轨音频
- **输出**: 最匹配的音源预设列表（Top 5）
- **匹配维度**: 乐器类别、音色特征（MFCC/频谱）、ADSR 包络
- **预设来源**: 内置预设库 + 用户可扩展

### F5 — 效果器参数预估 (Effects Parameter Estimation)
- **EQ 预估**: 频段中心频率、增益(dB)、Q 值
- **混响预估**: RT60、干湿比
- **压缩预估**: 阈值、压缩比、Attack/Release
- **输出格式**: JSON 参数文件，可导入 DAW

## 非功能需求

### N1 — 性能
- 4分钟立体声歌曲分轨 ≤ 5分钟 (GPU) / ≤ 20分钟 (CPU)
- MIDI 转录单轨 ≤ 30秒

### N2 — 可用性
- 紫色科技风 UI，简洁直观
- 支持项目保存/恢复，断点续做
- 中文界面优先

### N3 — 兼容性
- Windows 11 优先
- Python 3.13 兼容
- 支持 CUDA GPU 加速（可选）

### N4 — 扩展性
- 预设数据库支持用户添加
- 模块化设计，CLI 可独立运行
