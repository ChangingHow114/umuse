# UMuse — UI 设计规范

## 品牌标识

- **应用名称**: UMuse
- **定位**: AI 音乐逆向工程工作站
- **风格**: 深色科技风 (Dark Tech)

## 色彩系统

### 主色调 — 紫色系
```
Primary:    #7C3AED  (紫罗兰)  — 主按钮、选中态、Logo
Light:      #A78BFA  (浅紫)    — 悬停态、次要元素
Pale:       #C4B5FD  (淡紫)    — 边框、分隔线
```

### 背景色 — 深色系
```
BgDark:     #0F0A1A  (深紫黑)  — 主背景
BgPanel:    #1A1128  (面板色)  — 卡片、面板背景
BgSurface:  #231840  (表面色)  — 输入框、列表项
```

### 强调色 — 青色系
```
Accent:     #00F5FF  (电光青)  — 进度条、高亮、图表折线
AccentDim:  #0099AA  (暗青)    — 非活跃状态
```

### 功能色
```
Success:    #10B981  (绿色)    — 完成状态
Warning:    #F59E0B  (琥珀)    — 警告
Error:      #EF4444  (红色)    — 错误
```

## 字体

- **中文**: Microsoft YaHei (微软雅黑) → fallback: SimHei
- **英文/数字**: Segoe UI → fallback: Consolas (等宽, 代码/参数区)
- **字号层级**: 标题 18px / 副标题 14px / 正文 12px / 小字 10px

## 间距 & 圆角

- **基础单位**: 8px
- **页面内边距**: 24px
- **卡片间距**: 16px
- **元素间距**: 8px (小) / 16px (中) / 24px (大)
- **圆角**: 6px (控件) / 12px (卡片) / 20px (面板)

## 组件风格

- **按钮**: 圆角 8px，悬停上浮 1px 阴影，按下无阴影
- **输入框**: 深色背景 + 淡紫边框，聚焦时青色发光
- **进度条**: 青色渐变填充，圆角 4px
- **卡片**: 面板背景 + 1px 淡紫边框 + 8px 圆角
- **侧边栏**: 深紫黑背景，选中项紫色左边框指示器
- **音轨面板**: 波形显示 + 独奏(S)/静音(M) 切换
